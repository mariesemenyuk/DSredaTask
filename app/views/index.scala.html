REST API used car shop

This is a placeholder page to show the REST API.

Endpoints for Brand: 

      get all brands GET localhost:9000/brands
      add new brand POST localhost:9000/brands
      JSON: {
        "title": "",
        "country": ""
        }
      update brand PUT localhost:9000/brands/:id
      get brand GET localhost:9000/brands/:id
      delete brand DELETE localhost:9000/brands/:id

Endpoints for Model: 

      get all models GET localhost:9000/models
      add new model POST localhost:9000/models
      JSON:{
        "title": "",
        "productionStarted": int,
        "productionFinished": int
        }
      update model PUT localhost:9000/models/:id
      get model GET localhost:9000/models/:id
      delete model DELETE localhost:9000/models/:id

Endpoints for Shop Positions: 

      get all positions GET localhost:9000/shop
      get all positions of specific brand GET localhost:9000/shop/brand/:id
      add new position POST localhost:9000/shop
      JSON:{
        "brand_id": int,
        "model_id": int,
        "mileage": int,
        "price": int,
        "year_prod": int
        }
      update position PUT localhost:9000/shop/:id
      get position GET localhost:9000/shop/:id
      delete position DELETE localhost:9000/shop/:id
      get all positions of specific brand and model GET localhost:9000/shop/:brand/:model
